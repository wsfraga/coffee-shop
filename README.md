# ☕ Aurora Coffee Shop

Projeto pessoal de interface de loja de café com funcionalidades de busca, filtro e ordenação de produtos.

## 🚀 Funcionalidades

* 🔍 Busca de produtos por nome
* 💰 Filtro de produtos caros
* 📊 Ordenação por:

  * Menor preço
  * Maior preço
  * Nome (A-Z)
  * Nome (Z-A)

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript

## 🧠 Aprendizados

* Manipulação de arrays (`filter`, `sort`, `toSorted`)
* Organização de fluxo de dados
* Uso de eventos no DOM
* Separação de lógica e renderização

## 📂 Como rodar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/aurora-coffee-shop.git
```

2. Abra o arquivo `index.html` no navegador
